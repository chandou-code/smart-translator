# 智能翻译助手 - 项目说明

## 项目概述

这是一个基于 UniApp 的智能翻译助手应用，支持关键词汇标注、智能断句、语音朗读等功能。**纯前端实现，无需后端服务即可运行。**

## 功能特性

### 1. 关键词汇标注
- 用户可以粘贴包含括号翻译的英文文本
- 例如：`expanded（拓展）` 或 `expanded(拓展)`
- 系统会自动识别并隐藏翻译
- 点击带下划线的关键词汇可显示/隐藏翻译

### 2. 智能断句
- 长按文本时，系统会自动选中完整的句子
- 支持中英文标点符号识别（句号、逗号等）

### 3. 自定义长按菜单
- 长按句子会弹出自定义操作菜单
- 支持朗读和复制功能

### 4. 语音朗读
- 支持浏览器 TTS 和原生 TTS

## 运行

1. 使用 HBuilderX 打开 `翻译前端uniapp` 文件夹
2. 运行项目：
   - H5：运行 → 运行到浏览器 → Chrome
   - Android：运行 → 运行到手机或模拟器

## 技术栈

- **框架**: UniApp (Vue 3)
- **UI**: 现代卡片式设计，紫色渐变主题
- **TTS**: Web Speech API / 原生 TTS
